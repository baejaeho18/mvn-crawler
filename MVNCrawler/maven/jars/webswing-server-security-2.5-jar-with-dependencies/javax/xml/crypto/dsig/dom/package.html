DOM-specific classes for the {@link javax.xml.crypto.dsig} package. Only users who are using a DOM-based {@link javax.xml.crypto.dsig.XMLSignatureFactory XMLSignatureFactory} or {@link javax.xml.crypto.dsig.keyinfo.KeyInfoFactory} should need to make direct use of this package.
