Parameters of ColumnPruner¶

Affected Class¶

  • ai.h2o.sparkling.ml.features.ColumnPruner

Parameters¶

  • Each parameter has also a corresponding getter and setter method. (E.g.: label -> getLabel() , setLabel(...) )

columns

List of columns to be kept or removed

Scala default value: Array() ; Python default value: []

keep

Determines if the column specified in the ‘columns’ parameter should be kept or removed

Scala default value: false ; Python default value: False
