Sparkling Water Requirements¶

  • Linux/OS X/Windows

  • Java 1.8+

  • Python 3.6+ For Python version of Sparkling Water (PySparkling)

  • Spark 3.1 and SPARK_HOME shell variable must point to your local Spark installation
